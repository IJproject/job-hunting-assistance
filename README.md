### アプリ概要

自己分析から企業ごとのメモまで、就活での情報を一括で管理することができるアプリ

### 技術スタック

frontend: Vue, Vuetify, TypeScript, Vue-Test-Utils, Jest, StoryBook

backend: Laravel, Inertia, PHPUnit

infra: AWS(EC2, Route53), MySQL, Vite
